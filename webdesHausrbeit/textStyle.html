TextStyling

Intro

Text-Styling in CSS ermöglicht es Texte auf einer Webseite ansprechend und lesbar zu gestalten.

Damit können Schriftart, Schriftgröße, Schriftfarbe, Schriftstil und Textausrichtung angepasst werden.

So kann auf Webseiten der Text jeweils individuell angepasst werden um diesen so, je nach Bedürfnis, darstellen zu können.

color
background-color

Die Farbgestaltung von Texten und Hintergründen spielt in der Webentwicklung eine zentrale Rolle. Die CSS-Eigenschaften color und background-color ermöglichen diese Anpassung. Hier einige Möglichkeiten, Farben in CSS anzugeben:

  • Color-Keywords: z.B. red, blue, green, Mozilla Color List
  • RGB-Werte als HEX: z.B. #FF0000 (Rot)
  • RGB-Funktion: z.B. rgb(255, 0, 0)
  • RGBA-Funktion: z.B. rgba(255, 0, 0, 0.5) (halbdurchsichtig)
  • HSL-Funktion: für Farbton, Sättigung und Helligkeit
  • HSLA-Funktion: wie HSL, aber mit Transparenz




Mit background-color: blue; und color: rgb(255, 255, 255);

Mit Background-Color: blue; wird der Hintergrund blau gefärbt.

HTML-Code: 

                        <p class="color_bsp">
                            Mit <code>Background-Color: blue;</code> wird der Hintergrund blau gefärbt.
                        </p>

CSS-Code: 

                        .color_bsp{
                            color: rgb(255, 255, 255);
                        }
                        section:has(p.color_bsp){
                            background-color: blue;
                        }


Mit background-color: blue; und color: rgb(255, 255, 255);

Mit Color: rgb(255, 255, 255); wird die Schriftfarbe zu Weiß geändert.

HTML-Code: 

                        <p class="color_bsp">
                            Mit <code<Color: rgb(255, 255, 255);</code< wird die Schriftfarbe zu Weiß geändert.
                        </p>

CSS-Code: 

                        .color_bsp{
                            color: rgb(255, 255, 255);
                        }
                        section:has(p.color_bsp){
                            background-color: blue;
                        }

font-family

Mit font-family kann die Schriftart geändert werden. Es können mehrere Schriftarten angegeben werden, wobei nur die erste verfügbare verwendet wird. Einige Standard-Schriftarten sind:

  • Arial
  • Courier New
  • Georgia
  • Times New Roman
  • und Verdana



Mit font-family: 'Courier New';

Beispieltext mit Courier New Schriftart.

HTML-Code: 

                        <p class="courier">
                            Beispieltext mit Courier New Schriftart.
                        </p>

CSS-Code: 

                        .courier {
                            font-family: 'Courier New';
                        }

default-fonts

  • Serif: Traditionell und formell mit dekorativen Strichen an den Buchstabenenden.
  • Sans-serif: Modern und sauber ohne dekorative Striche. Gut für digitale Inhalte.
  • Monospace: Feste Breite für jeden Buchstaben, oft für Programmierung.
  • Kursive: Fließende Handschrift, elegant und persönlich.
  • Fantasy: Kreativ und dekorativ, für spezielle Themen.



Mit font-family: 'Times New Roman', Times, serif;

Beispieltext mit Serif-Schriftart.

HTML-Code: 

                        <p class="serif">

                        </p>

CSS-Code: 

                        .serif {
                            font-family: 'Times New Roman', Times, serif;

font-size

Mit font-size kann man die Größe der Schrift verändern, hierbei gibt es mehrere Möglichkeiten:

  • px - Pixel
  • em - So groß wie ein großes M im Eltern-Element groß ist
  • rem - So groß wie ein großes M im Root-Element (html) groß ist



Mit font-size: 24px;

Text mit 24 Pixel Schriftgröße.

HTML-Code: 

                        <p class="fontPx">
                            Text mit 24 Pixel Schriftgröße.
                        </p>

CSS-Code: 

                        .fontPx {
                            font-size: 24px;
                        }


Mit font-size: 1.5em;

Text hat 1.5em Schriftgröße und Elternelement 16 Pixel.

HTML-Code: 

                        <p class="fontEm">
                            Text hat 1.5em Schriftgröße und Elternelement 16 Pixel.
                        </p>

CSS-Code: 

                        section:has(p.fontEm) {
                            font-size: 16px;
                        }
                        .fontEm{
                            font-size: 1.5em;
                        }


Mit font-size: 1.5rem;

Text hat 1.5rem Schriftgröße und Root-Element 20px.

HTML-Code: 

                        <p class="fontRem">
                            Text hat 1.5rem Schriftgröße und Root-Element 20px.
                        </p>

CSS-Code: 

                        .fontRem {
                            font-size: 1.5rem;
                        }

font-style

Es gibt drei verschiedene Font-Style:

  • normal - normale Schrift, default
  • italic (Kursiv) font-style: italic; - wird verwendet, um den Text kursiv darzustellen. Dies bezieht sich auf eine spezifische kursive Schriftartvariante.
  • oblique (Schräg) font-style: oblique; - wird verwendet, um den Text schräg darzustellen. Im Gegensatz zu italic neigt oblique die Buchstaben in normalen Schriftarten ohne eine spezielle kursive Variante.



Mit font-style: italic;

Kursiver Text.

HTML-Code: 

                        <p class="fontItalic">
                            Kursiver Text.
                        </p>

CSS-Code: 

                        .fontItalic {
                            font-style: italic;
                        }


Mit font-style: oblique;

Schräger Text.

HTML-Code: 

                        <p class="fontOblique">
                            Schräger Text.
                        </p>

CSS-Code: 

                        .fontOblique {
                            font-style: oblique;
                        }

text-transform

Mithilfe von text-transform können z.B. ganze <p> Tags groß geschrieben werden.



Mit text-transform: uppercase;

alles innerhalb dieses Paragraphen wurde klein geschrieben und mit text-transform: uppercase; wird alles in caps geändert.

HTML-Code: 

                        <p class="uppercase">
                            alles innerhalb dieses Paragraphen wurde klein geschrieben und mit
                            <code>text-transform: uppercase;</code> wird alles in caps geändert.
                        </p>

CSS-Code: 

                        .uppercase {
                            text-transform: uppercase;
                        }


Mit text-transform: capitalize;

alles innerhalb dieses Paragraphen wurde klein geschrieben und mit text-transform: capitalize; wird jeder erste buchstabe jedes wortes großgeschrieben.

HTML-Code: 

                        <p class="capitalize">
                            alles innerhalb dieses Paragraphen wurde klein geschrieben und mit
                            <code>text-transform: capitalize;</code> wird jeder erste buchstabe jedes wortes großgeschrieben.
                        </p>

CSS-Code: 

                        .capitalize {
                            text-transform: capitalize;
                        }


Mit text-transform: lowercase;

ALLES INNERHALB DIESES PARAGRAPHEN WURDE GROß GESCHRIEBEN UND MIT text-transform: lowercase; WIRD ALLES IN LOWERCASE GEÄNDERT.

HTML-Code: 

                        <p class="lowercase">
                            ALLES INNERHALB DIESES PARAGRAPHEN WURDE GROß GESCHRIEBEN UND MIT
                            <code>text-transform: lowercase;</code> WIRD ALLES IN LOWERCASE GEÄNDERT.
                        </p>

CSS-Code:

                            text-transform: lowercase;

text-decoration

Mit text-decoration kann man eine Linie über, unter oder durch einen Text ziehen. Diese drei Möglichkeiten lassen sich auch miteinander kombinieren und mit Farben und Linienart verändern. Es gibt folgene Möglichkeiten

  • overline - Linie über dem Text
  • underline - Linie unter dem Text
  • line-through - Linie durch den Text
Zusätzlich gibt es diese Linienarten:
  • solid - normale Linie
  • double - doppelte Linie
  • dashed - gestrichelte Linie
  • dotted - gepunktete Linie
  • wavy - gewellte Linie



Mit text-decoration: underline;

Unterstrichener Text

mit text-decoration: underline;

HTML-Code: 

                         <h3 class="underline">
                            Unterstrichener Text
                         </h3>
                         <p>
                            mit  <code>text-decoration: underline; </code>
                         </p>

CSS-Code: 

                        .underline {
                            text-decoration: underline;
                        }

text-shadow

text-shadow gibt dem Text, wie der Name schon sagt, einen Schatten. Dieser benötigt folgende Werte in Reihenfolge:

  1. Horizontaler-Offset: positiv nach rechts, negativ nach links.
  2. Vertikaler-Offset: positiv nach unten, negativ nach oben.
  3. Blur-Radius: optional, je größer, desto unschärfer.
  4. Basis-Farbe: in verschiedenen Formaten wie Hex, RGB, RGBA, HSL.



Mit text-shadow: 2px 2px rgb(120, 120, 120);

text-shadow: 2px 2px rgb(120, 120, 120); - Schatten wird nach unten rechts geworfen

HTML-Code: 

                        <p class="botRight">
                            <code>text-shadow: 2px 2px rgb(120, 120, 120);</code> - Schatten wird nach unten rechts geworfen
                        </p>

CSS-Code: 

                        .botRight {
                            text-shadow: 2px 2px rgb(120, 120, 120);
                        }


Mit text-shadow: -2px -2px 2px rgb(120, 120, 120);

text-shadow: -2px -2px 2px rgb(120, 120, 120); - Schatten wird nach oben links geworfen und ist etwas unscharf

HTML-Code: 

                        <p class="topLeft">
                            <code>text-shadow: -2px -2px 2px rgb(120, 120, 120);</code> - Schatten wird nach oben links geworfen
                            und ist etwas unscharf
                        </p>

CSS-Code: 

                        .topLeft {
                            text-shadow: -2px -2px 2px rgb(120, 120, 120);
                        }
