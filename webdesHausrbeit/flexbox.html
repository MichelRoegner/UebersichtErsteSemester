FlexBox

Intro

Flexbox ist eine Layout-Methode um auf einer Website Elemente anzuordnen.


Dies ermöglicht simple responsive Layouts zu erstellen, indem Elemente beliebig auf der horizontalen und vertikalen Achse angeordnet werden können.


Weiter ist es möglich die Anordnung umzukehren oder auch einzelne Elemente beliebig zu verschieben.

flex

In dem Eltern-Element wird das mittels display:flex; die Art der Ansicht geändert. Ohne weiteren Zusatz werden die Kind-Elemente jetzt nebeneinander angezeigt

justify-content

Mittels justify-content werden die Kind-Elemente auf der gewählten Achse ausgerichtet (row and der Y-Achse und column an der X-Achse). Mit start oder flex-start werden alle Kind-Elemente an den Start der Hauptachse zusammengeschoben. Bei end bzw. flex-end werden diese am Ende der Hauptachse zusammengeschoben. center schiebt die Elemente zusammen und positioniert sie in der Mitte des Eltern-Elements. space-around teilt den freien Platz im Eltern-Element auf und setzt diesen an jedes Kind-Element auf BEIDEN Seiten. space-between setzt das erste Kind-Element auf den Start Achse und das letzte auf das Ende. Die übrigen Elemente werden bei gleichbleibenden Abstand dazwischen angeordnet. space-evenly verteilt den freien Platz auch auf den Start und das Ende des Eltern-Elements.

align-self

Mit align-self können Kind-Elemente unabhängig von deren Geschwister-Elementen ausgerichtet werden. Im Gegesatz zu der vorherigen Ausrichtungsarten, wird align-self auf die Kind-Elemente selber angewandt. align-self: stretch; streckt das Kind-Element auf die maximal mögliche Größe. align-self: center; zentriert das jeweilige Kind-Element. align-self: start; positioniert es am Anfang und align-self: end am Ende. align-self: baseline; richtet das Kind-Element an der Grundlinie aus.
