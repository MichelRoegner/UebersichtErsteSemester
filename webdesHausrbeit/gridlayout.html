Grid

Intro

Das CSS Grid Layout ist eine Technik zur Erstellung von Webseitenlayouts. Es ermöglicht die Erstellung eines zweidimensionalen Rastersystems, das eine präzise Positionierung von HTML-Elementen auf einer Webseite ermöglicht. Mit Grid können Entwickler komplexe Layouts erstellen, die flexibel, responsiv und einfach zu handhaben sind. Durch die Strukturierung von Inhalten in Spalten und Zeilen bietet das Grid Layout eine effiziente Möglichkeit, das Design und die Struktur von Webseiten zu gestalten, wodurch eine verbesserte Kontrolle über das Erscheinungsbild der Webseite erreicht wird.

grid

In diesem Element wird einmal die Anordnung der Inhalte mit und einmal ohne display: grid; Elemente gezeigt


Mit display:grid;

1

2

3

4

HTML-Code: 

                        <section class="grid">
                            <p>1</p>
                            <p>2</p>
                            <p>3</p>
                            <p>4</p>
                        </section>

CSS-Code: 

                        section.grid {
                            display: grid;
                            grid-template-columns: repeat(3, 1fr);
                            grid-template-rows: repeat(3, 1fr);
                        }
                        section.grid > p:nth-of-type(1) {
                            grid-row: 1 / 2;
                            grid-column: 1 / 3;
                        }
                        section.grid > p:nth-of-type(2) {
                            grid-row: 1 / 3;
                            grid-column: 3 / 4;
                        }
                        section.grid > p:nth-of-type(3) {
                            grid-row: 2 / 4;
                            grid-column: 1 / 2;
                        }
                        section.grid > p:nth-of-type(4) {
                            grid-row: 3 / 4;
                            grid-column: 2 / 4;
                        }

place-items

Im Normalfall füllt ein Element seine Zelle vollständig aus, dies lässt sich mit align-items für die Spalten Und justify-items für die Zeilen ändern.

Mit place-content und place-items lassen sich die Elemente in beide Richtungen gleichzeitig ausrichten.


Mit align-items: center; und justify-items: start;

1

2

3

4

HTML-Code:

                            <p>1</p>
                            <p>2</p>
                            <p>3</p>
                            <p>4</p>
                        </section>

CSS-Code: 

                        section.alignItems{
                            margin: 0;
                            display: grid;
                            grid-template-columns: 120px 120px;
                            grid-template-rows: 120px 120px;
                            align-items: center;
                            justify-items: start;
                        }
                        section.alignItems > p {
                            margin: 0;
                        }
